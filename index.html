Trainnotifier.co.uk

About

This project is aimed at using the Network Rail Data Feeds to help learn some new technologies through an area I am interested in.
I originally intended to do something with the Live Departure Board Web Service until National Rail made it pay-only.

Web Client Technologies Used

The web client uses:

  • HTML 5 based around the twitter bootstrap template
  • websockets to live track trains
  • Google Maps API to display maps
  • is hosted on windows azure web sites (free model at present)

Server Technologies Used

The server side uses:

  • Alchemy Websockets to handle websocket connections
  • Apache NMS Stomp (slighlty modified) to connect to the rail data feeds
  • ASP.NET MVC Web API for serving data to the web client
  • MySQL legacy database (stored on old web host)
  • SQL Server 2008 R2
  • Server components are hosted on Hyper-V virtual machines on my HP MicroServer
